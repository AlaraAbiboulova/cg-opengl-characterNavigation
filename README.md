#CG OpenGL Character navigation project

This project demonstrates character navigation in a 3D environment using OpenGL.  
It includes models, textures, and navigation logic for a simple scene.  

Features include:  
- Using **ray tracing** to choose a character for navigation  
- **Change textures** of the couch by clicking **"C"**  
- **Day/night toggle** through **"N"**  
- **Turn on the light** by clicking **"L"**  
- Character movement with **WASD** keys
------------------------

#Setup / Requirements

This project was developed and tested on Mac using VS Code.  
Includes following dependencies:

- OpenGL
- GLFW
- GLM
